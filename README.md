# CLA Signatures

This branch is dedicated to storing Contributor License Agreement signatures
for [Maestro Deck](https://github.com/blueshork/maestro-deck), managed by
[CLA Assistant](https://github.com/contributor-assistant/github-action).

**Do not protect this branch** — the bot needs direct write access to append
new signatures to `signatures/version1/cla.json`.

**Do not merge this branch into `main`** — signatures are operational data,
not source code. They live here permanently and independently.

The CLA text itself lives in [`CLA.md`](https://github.com/blueshork/maestro-deck/blob/main/CLA.md)
on the `main` branch.
